# Claude Code 期間限定プロモーション - 最大 1,000ドル の無料クレジット

## 概要

Anthropic は、Claude Pro および Claude Max ユーザー向けに、Claude Code（Web / モバイル版）で使える期間限定の無料クレジットプロモーションを実施しています。このプロモーションでは、**Pro ユーザーには 250ドル相当、Max ユーザーには 1,000ドル相当**の追加クレジットが提供されます。

## プロモーション期間

**2025年11月4日 9:00 AM PT 〜 2025年11月18日 11:59 PM PT**

期間終了後、未使用分のクレジットは失効します。

## 対象ユーザー

### 対象となる方
- **Claude Pro** または **Claude Max** の個人ユーザー
- プロモーション期間中に新規加入したユーザーも対象

### 対象外となる方
- Claude Team プランのユーザー
- Claude Enterprise プランのユーザー

## クレジット内容

### Claude Pro ユーザー
- **250ドル相当**の追加クレジット

### Claude Max ユーザー
- **1,000ドル相当**の追加クレジット

### クレジットの特徴
- API 料金換算でのクレジット付与
- **Claude Code（Web / モバイル版）専用**
- 通常の使用量とは別枠で管理
- クレジットが先に消費され、使い切った後は通常の Pro / Max 利用枠に自動切り替え
- **追加請求は発生しません**

## 利用方法

### 1. Claude Code にアクセス

ブラウザで以下の URL にアクセスしてください：

```
https://claude.ai/code
```

### 2. GitHub アプリ連携

Claude Code を使用するには、GitHub アプリとの連携が必要です。初回アクセス時に連携の設定が求められます。

### 3. クレジットの確認

- クレジットは自動的に適用されます
- UI 上で現在の残高を確認できます
- 使用状況をリアルタイムで追跡可能

## 利用上の注意事項

### クレジットの制限事項
- ⚠️ **現金価値はありません**
- ⚠️ **他の用途への振替不可**
- ⚠️ **他のキャンペーンとの併用不可**
- ⚠️ **第三者への譲渡不可**
- ⚠️ **数量限定**（先着順の可能性があります）

### クレジット消費について
1. まず、プロモーションクレジットから消費されます
2. クレジットを使い切ると、通常の Pro / Max の利用枠に自動的に切り替わります
3. プロモーション期間終了時点で、未使用のクレジットは失効します

## Claude Code でできること

このクレジットを使って、Claude Code の強力な機能を存分に活用できます：

### コーディング支援
- コードの作成・編集・リファクタリング
- バグの特定と修正
- コードレビューとベストプラクティスの提案

### プロジェクト管理
- Git 統合による変更管理
- プルリクエストの作成とレビュー
- コミットメッセージの自動生成

### 開発ワークフロー
- カスタムコマンドによる自動化

## よくある質問

### Q: クレジットはいつ付与されますか？
A: 対象ユーザーには自動的に付与され、Claude Code にアクセスすると確認できます。

### Q: クレジットを使い切ったらどうなりますか？
A: 自動的に通常の Claude Pro / Max の利用枠に切り替わります。追加料金は発生しません。

### Q: Web 版以外の Claude でクレジットを使えますか？
A: いいえ。このクレジットは **Claude Code（Web / モバイル版）専用**です。

### Q: プロモーション期間後はどうなりますか？
A: 未使用のクレジットは失効します。ただし、通常の Claude Pro / Max のサブスクリプションは継続されます。

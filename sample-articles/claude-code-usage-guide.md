# Claude Code の使い方

## 概要

Claude Code は、Anthropic が提供する公式 CLI ツールで、ソフトウェア開発タスクを支援するインタラクティブなコマンドラインインターフェースです。このガイドでは、Claude Code の基本的な使い方と主要な機能について説明します。

## 主な機能

### 1. ファイル操作

Claude Code は、以下のようなファイル操作を効率的に実行できます：

- **ファイルの読み込み**: プロジェクト内のファイルを読み取り、内容を理解します
- **ファイルの編集**: 既存のファイルを編集し、コードの改善や修正を行います
- **ファイルの作成**: 新しいファイルやディレクトリを作成します

### 2. コード検索と分析

- **Glob パターン検索**: `**/*.js` のようなパターンでファイルを検索
- **Grep 検索**: コード内のキーワードや正規表現パターンを検索
- **コードベースの探索**: プロジェクト構造の理解と関連コードの特定

### 3. Git 統合

Claude Code は Git と緊密に統合されており、以下の操作をサポートします：

- コミットの作成と管理
- ブランチの作成と切り替え
- プルリクエストの作成（gh CLI 経由）
- 変更履歴の確認

### 4. タスク管理

複雑なタスクを小さなステップに分解し、進捗を追跡します：

```
1. 要件の分析
2. 実装計画の作成
3. コードの実装
4. テストの実行
5. ドキュメントの更新
```

## 基本的な使い方

### 1. ファイルの編集を依頼する

```
「src/index.js のエラーハンドリングを改善してください」
```

Claude Code は、ファイルを読み込み、適切な改善を提案・実装します。

### 2. 新機能の実装

```
「ユーザー認証機能を追加してください」
```

Claude Code は、必要なファイルを作成し、機能を実装します。

### 3. コードレビュー

```
「このプルリクエストをレビューしてください」
```

コードの品質、潜在的な問題、改善提案を提供します。

### 4. デバッグ支援

```
「テストが失敗しています。原因を特定して修正してください」
```

エラーログを分析し、問題を特定して修正します。

## カスタムコマンド

`.claude/commands/` ディレクトリにマークダウンファイルを配置することで、カスタムスラッシュコマンドを作成できます。

例：`.claude/commands/review.md`

```markdown
プルリクエストのコードレビューを実行してください：
1. コード品質のチェック
2. セキュリティ上の懸念の確認
3. パフォーマンスの最適化提案
4. テストカバレッジの確認
```

使用方法：
```
/review
```

## ベストプラクティス

### 1. 明確な指示を与える

具体的で明確な指示により、より良い結果が得られます：

❌ 悪い例: 「コードを直して」
✅ 良い例: 「src/auth.js のパスワード検証ロジックを、特殊文字を必須とするように変更してください」

### 2. コンテキストを提供する

関連する情報を提供することで、より適切な実装が可能になります：

```
「このプロジェクトは Express.js を使用しています。
/api/users エンドポイントに認証ミドルウェアを追加してください」
```

### 3. 段階的なアプローチ

大きなタスクは小さなステップに分解します：

```
1. まずユーザーモデルを作成
2. 次に認証ロジックを実装
3. 最後にテストを追加
```

## まとめ

Claude Code は、開発ワークフローを効率化し、コード品質を向上させる強力なツールです。基本的な使い方を理解し、カスタムコマンドを活用することで、開発生産性を大幅に向上させることができます。

質問や不明点があれば、いつでも Claude Code に尋ねてください！
